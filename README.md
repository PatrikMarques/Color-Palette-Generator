# 🎨 Color Palette Generator

Um gerador de paletas de cores aleatórias desenvolvido para praticar a manipulação do DOM, eventos em JavaScript e a API de Clipboard do navegador.

## 🚀 Funcionalidades
- Gerar 5 cores aleatórias com um clique.
- Copiar o código Hexadecimal da cor clicando no ícone ou na própria cor.
- Feedback visual ao copiar (ícone de check).
- Layout responsivo para dispositivos móveis.

## 🛠️ Tecnologias Utilizadas
- **HTML5**: Estruturação semântica.
- **CSS3**: Estilização com Flexbox, Grid e Gradientes.
- **JavaScript (Vanilla)**: Lógica de geração de cores, manipulação do DOM e funções assíncronas (Clipboard API).
- **Font Awesome**: Ícones de interface.
- **Google Fonts**: Tipografia Roboto.
